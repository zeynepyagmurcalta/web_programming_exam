# Web Programming Exam Project

This project is a sample web page for **Tech Solutions Inc.** team. It is built using **HTML, CSS, and Bootstrap 5**.

---

## Features

- **Navbar**: Sections for Home, Team, Contact, and language selection.
- **Team Section**:
  - Profile photos
  - Card layout with hover effects
  - Social media icons with gradient color
- **Contact Section**:
  - Contact Us modal window
  - Company information and contact details
- **Footer**: Fixed at the bottom of the page, showing company info and social icons
- **Language Support**: Turkish and English
